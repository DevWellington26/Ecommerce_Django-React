# How to install?

Make a python virtualenv:
```
virtualenv myenv
```
Then to activate the virtualenv:
```
source myenv/bin/activate
```
Then install dependencies:
```
pip install -r requirements.txt
```
Now you're able to run server:
```
python manage.py runserver
```
